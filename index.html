Proofreader → Helper2.0





What is this?

This is a tool to aid you in proofreading someone else's work. If you don't have permission to edit someone's file, or you simply want to tell them the errors that need to be fixed rather than fixing them yourself, you can use this tool to inform them of those errors in a nicely-formatted way. Yes, it's simple — but I think it may be faster than having to manually copy and paste whatever formatting you are using to display all the errors you've found.

How do I use it?

There are two types of "elements" in a Proofreader Helper message — sections and errors. This is reflected by the fact that there is a "Next error" button as well as a "Next section" button. If the piece of writing you are proofreading is very small, you may only need one section. If it contains many different slides, paragraphs, or any other clear sections, you may wish to add a section for each of those. After you've added a new section, you can type the name in the text box. When you've added a new error, type the original text on the left side, and the corrected text on the right side. After you have finished proofreading, click the "Finished" button. Then you can copy and paste the corrections in your preferred format.

Which formats does it provide?

Plain text, BBCode, Markdown, Wikitext, HTML, and JSON. If you have a suggestion for a format that I missed, please submit an issue on GitHub.
